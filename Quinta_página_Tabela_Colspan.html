Tabelas

Quando precisamos lidar com dados tabulares em uma página web utilizamos tabelas.
As tabelas usadas nesse documento são exemplos do tipo de dado e também da apresentação obtida ao utilizarmos tais elementos na HTML.

Estrutura das Tabelas

A marcação HTML relacionada às tabelas contém:
A tag principal que define o container geral < table >;
A tag para a definição de linhas < tr >;
A tag para a definição de colunas < td >.

Entretanto, há tags adicionais que podem ser usadas e que ajudam a melhor organizar o conteúdo.

O atributo usado para a expansão (mesclar) de linhas rowspan.
O atributo usado para a expansão (mesclar) de colunas colspan.
