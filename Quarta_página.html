LISTAS

O HTML fornece suporte para a representação visual de três tipos de listas: as ordenadas, as não ordenadas e as de definição.

Lista ordenada

Exemplo de como ficará a página usando tags para criar uma lista ordenada:

  1. Primeiro item da lista;
  2. Segundo item da lista;
  3. Terceiro item da lista.

Lista não ordenada

Exemplo de como ficará a página usando tags para criar uma lista NÃO ordenada:

  • Primeiro item da lista não ordenada;
  • Segundo item da lista não ordenada;
  • Terceiro item da lista não ordenada.

Lista de definição

Exemplo de como ficará a página usando tags para criar uma lista com definição:

Primeiro item da lista não ordenada;
Descrição da primeira definição.
Segundo item da lista não ordenada;
Descrição da segunda definição.
Terceiro item da lista não ordenada;
Descrição da terceira definição.
