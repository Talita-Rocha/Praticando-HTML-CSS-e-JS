CSS in line

Essa forma implica em declarar o estilo CSS diretamente na tag, no código HTML.
Veja a seguir um exemplo de um estilo apresentado anteriormente sendo aplicado de forma Inline.

Nossa primeira página html a gente nunca esquecerá!

A declaração Inline faz uso do atributo “style” procedido por declarações separadas por ponto-e-vírgula “;”.
Esse atributo pode ser usado em qualquer tag HTML.
